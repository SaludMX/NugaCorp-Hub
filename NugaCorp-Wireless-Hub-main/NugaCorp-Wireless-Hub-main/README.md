# NugaCorp Wireless Hub — Perfil Comercial

Regla inmutable: Este README no incluye información confidencial ni sensible de seguridad. Es exclusivamente un perfil comercial/ejecutivo del proyecto.


## Visión

Ser la plataforma integral de gestión para WISPs/ISPs que unifica operación, visibilidad y toma de decisiones, reduciendo fricción operativa y elevando la experiencia del cliente final.

## Misión

Empoderar a equipos técnicos y de negocio con un Hub seguro y escalable que centraliza inventario, clientes, redes y automatizaciones, garantizando eficiencia operativa, resiliencia y crecimiento sostenible.

## Objetivo

Entregar un producto multi‑tenant listo para producción que:
- Estandariza flujos clave (provisión, monitoreo, soporte).
- Reduce OPEX y tiempos de respuesta.
- Mejora uptime y calidad del servicio.
- Facilita expansión y gobernanza con buenas prácticas.

## Propuesta de Valor

- Operación Unificada: Inventario, clientes, tickets y redes en un solo lugar.
- Seguridad por Diseño: Multi‑tenant con aislamiento y controles de acceso.
- Automatización Controlada: Jobs de MikroTik vía infraestructura segura (mock en esta fase).
- Inteligencia Operacional: Dashboards y KPIs para decisiones ágiles.
- Escalabilidad: Arquitectura modular preparada para integrar servicios futuros.

## Público Objetivo

- WISPs en crecimiento que requieren eficiencia y orden.
- ISPs regionales con necesidad de estandarizar operaciones.
- Operadores municipales/empresariales con foco en calidad de servicio.

## Estado del Proyecto

- FASE 3: CRUDs y vistas principales completas.
- FASE 4A: Infraestructura de jobs para MikroTik (multi‑tenant, RLS) lista.
- Roadmap: Métricas de jobs, UI avanzada de filtros, auditoría.

## Inicio Rápido

Requisitos: Node.js

1. Instalar dependencias:
   `npm install`
2. Levantar entorno de desarrollo:
   `npm run dev`

Para más información, contacte al equipo de ingeniería.
